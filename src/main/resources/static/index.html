A2A and MCP Security Integration Demo

Live Demo Available!

This demo is deployed and can be accessed at:
https://vishalmysore-a2amcpdemo.hf.space/.well-known/agent.json

Test the Demo

Try these commands to see role-based access in action:

1. Test Agent Discovery

With USER role:

curl -u user:password https://vishalmysore-a2amcpdemo.hf.space/.well-known/agent.json

Shows user-level actions like booking cars and checking status.

With ADMIN role:

curl -u admin:admin https://vishalmysore-a2amcpdemo.hf.space/.well-known/agent.json

Shows additional admin capabilities like canceling bookings and maintenance operations.

Without authentication:

curl https://vishalmysore-a2amcpdemo.hf.space/.well-known/agent.json

Shows only public actions available to unauthenticated users.

2. Test MCP Tools

List available tools (no auth):

curl -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/list","params":{},"id":9}' \
  https://vishalmysore-a2amcpdemo.hf.space/

List tools with USER role:

curl -u user:password -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/list","params":{},"id":9}' \
  https://vishalmysore-a2amcpdemo.hf.space/

List tools with ADMIN role:

curl -u admin:admin -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/list","params":{},"id":9}' \
  https://vishalmysore-a2amcpdemo.hf.space/

3. Test Tool Execution

Call a tool (ADMIN example - cancel booking):

curl -u admin:admin -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "cancelCarBooking",
      "arguments": {
        "provideAllValuesInPlainEnglish": {
          "arg0": "2233"
        }
      }
    },
    "id": 25
  }' \
  https://vishalmysore-a2amcpdemo.hf.space/

Call a tool (USER example - book car):

curl -u user:password -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"bookCar","arguments":{"provideAllValuesInPlainEnglish":{"arg0":"sedan","arg1":"toronto","arg2":"vancouver"}}},"id":1}' \
  https://vishalmysore-a2amcpdemo.hf.space/

Features Demonstrated

  • Role-based access control (USER and ADMIN roles)
  • Annotation-based security (@PreAuthorize)
  • Dynamic agent card generation based on roles
  • Seamless integration of A2A and MCP protocols
  • Support for both Python and Java clients
